# McuPilot Setup — PyWebView 实现计划

## 技术栈

| 层 | 技术 | 用途 |
|----|------|------|
| 窗口容器 | `pywebview` | 原生 Win32 窗口，内嵌 Edge WebView2 |
| 前端 | HTML + CSS + vanilla JS | 界面渲染、动画、用户交互 |
| 前后端桥 | `webview.js_api` | JS → Python 互调用 |
| 后端 | Python + `core/project_wizard.py` | 环境检测、代码生成、部署编排 |
| 打包 | PyInstaller | 单 exe 输出 |

---

## 一、文件结构

```
McuPilot/
├── setup_gui.py                    # ← 入口（ PyWebView 窗口 + 后端桥接）
├── core/
│   └── project_wizard.py           # ← 后端逻辑（环境检测 + 部署编排）
├── assets/
│   └── setup_ui.html               # ← 前端 UI（单文件，HTML+CSS+JS 写一起）
└── ...
```

**单 HTML 文件的理由：** 不需要前后端分离、不需要静态资源服务器。一个 HTML 字符串嵌入 Python 或用 `file://` 加载。简单、打包方便。

## 二、实现顺序

### Phase 1：环境搭建（10 分钟）

1. `pip install pywebview`
2. 验证：写一个最小 demo `hello_webview.py`，打开窗口显示 HTML
3. 验证 js_api：JS 调 Python 弹出文件夹选择对话框

### Phase 2：前端 UI 静态稿（先不做后端）

写出完整的 HTML+CSS+JS 静态页面，包含：
- 页面一：工程设置（表单 + 参数表 + 客户端勾选）
- 页面二：部署进度
- 两页之间的切换动画
- 所有按钮、输入框、下拉的样式
- 此时没有真实数据，用假数据填充

### Phase 3：前端 → 后端桥接

- 用 js_api 把"浏览按钮"、"下一步"、"开始部署"等操作接到 Python 函数
- 把后端检测结果和部署进度推回 JS 更新 UI

### Phase 4：后端逻辑（复用 + 扩展）

- 把 `core/project_wizard.py` 写完整
- 启动检测（Python / pip）
- 环境检测（Keil / J-Link / MCU）
- 部署六步

### Phase 5：PyInstaller 打包 + 测试

## 五、Python↔JS 桥接设计

```python
class Api:
    """所有暴露给 JS 的方法"""
    
    def browse_folder(self) -> str:
        """弹出文件夹选择对话框，返回路径"""
        
    def on_path_changed(self, path: str) -> dict:
        """路径变化时触发环境检测，返回 {keil, jlink, mcu} 结果"""
        
    def next_to_page2(self) -> None:
        """JS 通知后端切换到页面二（后端准备摘要数据）"""

    def deploy(self) -> None:
        """开始部署，通过回调推送进度到 JS"""
        
    # 进度回调 → JS 通过 window.eval() 或 js_api 反向通知
```

JS 调 Python：
```javascript
// 点浏览按钮
const path = await window.pywebview.api.browse_folder()
document.getElementById('path-input').value = path
// 自动触发环境检测
const result = await window.pywebview.api.on_path_changed(path)
updateEnvStatus(result)
```

Python 推进度到 JS：
```python
def deploy(self):
    webview.windows[0].evaluate_js(f'updateStep(1, "success")')
    webview.windows[0].evaluate_js(f'setProgress(17)')
    # ...
```

## 六、窗口配置

```python
import webview

window = webview.create_window(
    title='McuPilot — 项目配置向导',
    url='assets/setup_ui.html',   # 或 html=HTML_STRING
    width=680,
    height=640,
    resizable=False,              # 固定尺寸，保持布局整洁
    confirm_close=True,           # 部署进行中时确认退出
)
webview.start(debug=False)
```
